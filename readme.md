## 介绍
本项目是根据[Python 练习册，每天一个小程序][pythond-examples]的个人练习，无他用。

## 基础环境
1. python version 3.6.3  
2. pip version 9.0.1  
``` cmd
pip install Pillow
pip install PyMysql
```
